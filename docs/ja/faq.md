# よくある質問 {#faq}

::: warning
**こちらのよくある質問は V4 バージョンのみ対象としています**

**V2 バージョンについては [よくある質問 (V2 アーカイブ)](/ja/v2/faq) をご覧ください**
:::

### 再生、検索、停止、プレイリストコマンドが利用不可と表示されるのはなぜですか？ {#why-are-music-commands-unavailable}

V4 では現在、基本コマンドと権限制御を利用できますが、音楽機能はまだ開発中です。

そのため、`/zmusic play`、`/zmusic search`、`/zmusic stop`、`/zmusic playlist` とそのサブコマンドは、現在は機能が利用できない旨を表示します。

### 現在使用できるコマンドは何ですか？ {#available-commands}

現在利用できるコマンドは次のとおりです：

- `/zmusic help [command]`
- `/zmusic info`
- `/zmusic reload`

ルートコマンドのエイリアスは `/music` と `/zm` です。

### サーバーにインターネット接続がない場合、ZMusic を使用できませんか？ {#no-network}

現在利用できる基本コマンドは、音楽 API へのネットワーク接続に依存しません。

今後、音楽再生、検索、歌詞表示、プレイリスト機能が完成した後は、関連機能で設定済みの音楽 API サービスへのアクセスが必要になります。

### V4 は NetEase Cloud Music ログインに対応していますか？ {#netease-login}

まだ対応していません。現在の V4 には `/zm login` または同等のログインコマンドはありません。

### NetEase API ドキュメントは現在何に使いますか？ {#netease-api-docs-purpose}

NetEase API ドキュメントは、デプロイ可能な外部 API サービスを説明するものです。V4 の音楽機能はまだこれらのサービスに接続されていないため、現在は今後の機能開発に向けた準備資料です。
