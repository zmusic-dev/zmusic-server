# 紹介

::: warning
**現在表示しているドキュメントは V4 バージョンです**<br>
**このバージョンはまだ開発中であり、エラーや不完全な部分がある可能性があります**<br>
**現在の長期サポートバージョンは V2 です。[V2 ドキュメント](/ja/v2/)をご覧ください**
:::

ZMusic V4 は現在開発中の Minecraft 音楽システムです。

現在実装済み：

- クロスプラットフォームのプラグイン基盤（Bukkit/Spigot/Paper/Folia、BungeeCord、Velocity）
- 基本コマンド、権限制御、Tab 補完
- 設定と i18n ファイルの読み込み
- 実行時依存関係の読み込み

開発中：

- 音楽再生と検索
- 全サーバーでのリクエスト
- 歌詞表示
- プレイリストシステム
- 再生キュー
