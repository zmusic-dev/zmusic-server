# 紹介

::: warning
**現在表示しているドキュメントは V4 バージョンです**<br>
**このバージョンはまだ開発中であり、エラーや不完全な部分がある可能性があります**<br>
**現在の長期サポートバージョンは V2 です。[V2 ドキュメント](/ja/v2/)をご覧ください**
:::

これは強力な音楽システムで、以下の機能をサポートしています。

- 音楽再生
- 全サーバーでのリクエスト
- 歌詞表示
- プレイリストシステム
- 再生キュー
- クロスプラットフォーム対応（Spigot/BungeeCord/Velocity）
