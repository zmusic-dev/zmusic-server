# 介绍 {#introduction}

::: warning
**您当前正在查看的文档为 V4 版本文档**<br>
**此版本仍在开发中，可能存在错误或不完善的地方**<br>
**当前长期支持版本为 V2，您可以查看 [V2 文档](/v2/)**
:::

ZMusic V4 是正在开发中的 Minecraft 音乐系统。

当前已完成：

- 跨平台插件基础框架（Bukkit/Spigot/Paper/Folia、BungeeCord、Velocity）
- 基础命令、权限控制和 Tab 补全
- 配置与国际化文件加载
- 运行时依赖加载

正在开发中：

- 音乐播放与搜索
- 全服点歌
- 歌词显示
- 歌单系统
- 播放队列
